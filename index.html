Finally, an usable Librus client.
Librusik is a Librus Synergia web client with many excellent features. Access your school things quickly and with ease. Forget about your weird login. Forget even about logging in, as Librusik will remember you. Forget about all school-defined limitations and go beyond them.

Features
Librusik provides a bunch of great features. Those include independent average calculation and detailed attendances report for each subject.
Fast & lightweight
Librusik backend can run even on the weakest hardware. User interface is designed to be simple and easy yet fairly functional.
For daily use
Librusik with its features is strong enough to be used as a daily driver. Project created in 2019 was quite stable and had only few major issues.
Multi-user
One instance is able to handle up to 128 users, so your friends or family won't need any knowledge how to self-host Librusik.
